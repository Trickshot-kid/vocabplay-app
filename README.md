# Vocabplay — Synonyms & Antonyms Classroom App

A fully self-contained browser-based educational application 
designed for classroom use.

## Features
- 4 interactive activities (Card Match, Fill in the Blank, 
  Word Web Builder, Opposite Day)
- 10-question adaptive quiz with graded feedback
- Gamified points system
- 4-week milestone progress tracker
- Persistent localStorage for saving progress
- Fully responsive — works on desktop, tablet, and mobile

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- localStorage API

## How to Run
Simply open `index.html` in any modern browser.
No installation or dependencies required.
